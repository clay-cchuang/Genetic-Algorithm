# 多目標基因算法實作

分享簡易多目標GA算法的實作，多目標採用權重來分配各項目標的重要性。
多目標GA的優勢在於可以根據不同情境進行排程情境的調換，以本題為例，多目標GA可在最小化完工時間、最小化整備時間與最小化延遲時間中取得平衡。
